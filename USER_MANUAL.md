# VocabForge 用户手册

## 简介

VocabForge 是一款通过双语台词学习单词的工具。上传中英文台词对 + 单词列表，自动生成带高亮标记和学习思考的结构化笔记，支持 Markdown 和 PDF 导出。

**访问地址：** https://frontend-seven-tau-48.vercel.app

---

## 快速开始

### 方式一：加载示例数据（推荐首次使用）

1. 打开首页，点击 **加载示例数据** 按钮
2. 系统自动填入《疑犯追踪 S01E01》台词 + 40+ 单词
3. 点击 **生成关联**
4. 等待几秒，右侧即显示学习笔记

### 方式二：使用自己的素材

1. **输入台词本**
   - 粘贴中英文台词（每行一对，英文在上中文在下）
   - 或上传文件（支持 PDF / Word / Excel / TXT / MD）
2. **输入单词列表**
   - 粘贴单词（每行一个）
   - 或上传文件
3. 点击 **生成关联**
4. 查看结果并下载

---

## 输入格式要求

### 台词本格式

英文和中文各占一行，交替排列：

```
When you find that one person who connects you to the world...
当你找到你在这个世上的羁绊你就变了变得更好 00:08
When that person is taken from you...
而当这个人从你身边被夺走 00:21
what do you become then?
那你又会变得怎样 00:27
```

> 中文行末尾的时间戳（如 `00:08`）可选，工具会保留显示。

### 单词列表格式

每行一个单词：

```
connect
whole
trust
private
purpose
```

## 结果说明

### 学习笔记内容

生成的结果为双栏布局：

| 左栏：台词 | 右栏：单词 |
|:---|:---|
| 英文台词（匹配的单词**黄色高亮**）<br>中文翻译 + 时间戳 | 单词 / 音标 / 词性 / 释义<br>💡 **AI 教师思考**（用法、搭配、剧情关联） |

### 下载

- **下载 PDF**：点击后浏览器弹出打印窗口，选择"另存为 PDF"。包含页码、紧凑排版
- **下载 MD**：直接下载 Markdown 文件，可在 Obsidian 等工具中打开

## 常见问题

### 后端休眠怎么办？

Render 免费方案超过 15 分钟无人访问会自动休眠。如果遇到加载超时，**刷新页面等 30-60 秒**即可，后端会自动唤醒。

### 文件上传失败？

- 确认文件格式在支持列表中
- PDF 文件不要超过 50 页
- 如果持续失败，尝试直接粘贴文本

### 结果不对？

- 检查台词格式是否正确（英文/中文交替）
- 检查单词是否每行一个
- 中文台词末尾可加时间戳，不影响匹配

### 如何获取台词和单词？

- 字幕网站（如 Sub HD、字幕库）下载双语字幕
- 复制台词中的英文对白，删除时间轴代码
- 单词可以从文章、教材或单词书中提取
